
# Cricket Score Management System

## Overview

Welcome to the **Cricket Score Management System**, a dynamic project developed in **Core Java** with a user-friendly Graphical User Interface (GUI) powered by **JavaSwing**. The project incorporates a plethora of JavaSwing components, such as JFrame, Container, JLabel, JTextField, JTextArea, Font, Color, JPanel, JTable, JSpinner, ImageIcon, Icon, JButton, and ActionListener, to create an interactive and engaging user experience.

Key Components-:

- **GUI Elements:**
  - JFrame
  - Container
  - JLabel
  - JTextField
  - JTextArea
  - Font
  - Color
  - JPanel
  - JTable
  - JSpinner
  - ImageIcon
  - Icon
  - JButton
  - ActionListener

- **OOP Concepts:**
  - Inheritance
  - Polymorphism

- **Core Concepts:**
  - Exception Handling
  - Multithreading


 Project Authorship -:

This project was meticulously crafted by a single developer within a 2-week timeframe.


Project Highlights -:

1. User-Friendly Interface

Enjoy an intuitive and visually appealing interface, carefully crafted to enhance user experience and interaction.

2. Object-Oriented Prowess

The codebase is designed with a strong emphasis on Object-Oriented Programming (OOP) principles, including inheritance and polymorphism. This ensures a scalable, maintainable, and extensible foundation for future enhancements.

3. Robust Core Functionality

- **Exception Handling:** Experience seamless error management with implemented exception handling for graceful handling of unexpected scenarios.

- **Multithreading:** Boost performance through the implementation of multithreading, allowing concurrent execution of tasks for improved efficiency.
